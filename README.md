# Testing-Browser-Controller

## Description

This is a convenient web view controller, and support to open multiple size windows at the same time. User can use to track web site on different devices more convenient.

### Support devices

   - iPhone4
   - iPhone5
   - iPhone6
   - iPhone6 +
   - iPad mini
   - Google Nexus 4
   - Google Nexus 7
   - Google Nexus 10
   - Samsung S4
